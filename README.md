

# 見だ (Mida) | 日语单词记忆 🌸

**見だ (Mida)** 是一款专为日语学习者打造的现代化单词记忆 Web 应用。它结合了极简主义的美学设计、目前最先进的 **FSRS (Free Spaced Repetition Scheduler)** 记忆算法，以及强大的 **DeepSeek AI** 智能辅助，旨在为你提供最高效、最无压力的日语单词背诵体验。

---

## ✨ 核心功能亮点

### 🧠 1. 基于 FSRS 的科学记忆算法
应用完全摒弃了传统的固定时间复习法，底层接入了先进的 `ts-fsrs` 引擎。
* **个性化调度**：每次复习时，提供 1(重来)、2(困难)、3(良好)、4(简单) 四个选项，算法会自动计算出下一次最佳复习时间。
* **记忆冷却池**：处理分钟级（如 1 分钟、10 分钟）的短期记忆时，单词会进入“冷却池”，到期后精准推送，确保形成初始记忆。
* **深度自定义**：支持用户在后台调节**目标记忆率 (Request Retention)**、最大复习间隔、每日新词上限、以及开启间隔随机化 (Fuzz)。
* **无痕巩固模式 (Cram Mode)**：“再练一次”功能提供纯净的复习体验，既能满足突击需求，又不会污染真实的 FSRS 数据模型。

### 🤖 2. DeepSeek AI 深度赋能
录入单词太麻烦？我们接入了 DeepSeek 大语言模型，极大降低了词库构建的门槛。
* **单次智能补全**：只需输入日语汉字（如“桜”），点击“AI 补全”，即可自动生成假名读音、中文含义、词性及相关标签（如 N5、日常）。
* **批量长文本解析**：直接粘贴一整段日文文章或单词表，AI 会自动提取有效单词、去重，并生成完整的属性信息，实现一键批量导入。

### 🃏 3. 沉浸式 3D 卡片交互
* **物理级撕纸动效**：学习界面的卡片采用了 3D 层叠设计 (Transform 3D)，点击揭晓答案时，卡片正面会呈现拟真的“撕下便签”飞走效果，带有阻尼感与空气阴影，让背单词变成一种享受。
* **全键盘操作**：支持使用 Space / Enter 翻开卡片，数字键 1-4 直接提交复习反馈，无需鼠标即可行云流水地完成学习。

### 🗂️ 4. 强大的词库管理与搜索
* **罗马音模糊检索**：搜索框内置了日文假名与罗马音的映射算法。即使你忘记怎么打出日文，直接输入罗马音（如 `sakura`）也能精准搜出“桜/さくら”。
* **批量操作与导入导出**：支持多选批量删除；支持将选中单词或整个词库导出为 JSON / CSV 格式。
* **防重复机制**：批量导入时，系统会自动与数据库已有单词比对并去重，防止脏数据污染词库。

### 🔔 5. 多维度复习提醒机制
实时轮询后端数据，当有单词到期需要复习时，支持以下提醒方式：
* 浏览器桌面推送通知 (Push Notifications)。
* 柔和的风铃音效 (C6/E6 泛音和弦)。
* 移动端物理震动反馈。
* 页面标题动态闪烁防错过。

## 💡 为什么开发見だ (Motivation）

市面上已经有很多背单词软件了，为什么我还要自己动手做一个？

在学习日语的过程中，我发现现有的工具总是有这样或那样的痛点：
1. **过于臃肿**：我只想要一个安静背单词的地方。
2. **死板的复习机制**：有些软件的算法不够聪明，让我每天被海量的复习任务淹没，或者反复复习早已掌握的单词。
3. **都不如anki**：所以我抄的anki，极度丐版，仅符合我本人的所有需求。

所以我选择自己做一个，只为打造一个纯粹、安静、高度沉浸的单词卡片机，把最高效的 40 分钟还给自己。

也许我们并不需要碎片化


## 🛠️ 技术栈

* **前端**: 原生 HTML5 / JavaScript (ES6+), Tailwind CSS (深度定制动画与质感)
* **后端**: Node.js, Express
* **数据库**: MongoDB (Mongoose)
* **核心算法库**: `ts-fsrs`
* **安全/鉴权**: JWT (JSON Web Tokens), bcryptjs, 针对 AI 接口的速率限制 (Rate Limiting)

## 🚀 部署与运行

### 环境准备
确保你的服务器已安装 Node.js (v18+) 和 MongoDB。

### 1. 克隆与安装依赖
```bash
git clone [https://github.com/taiyanghua258/mida-vocab.git](https://github.com/taiyanghua258/mida-vocab.git)
cd mida-vocab/backend
npm install
### 2. 配置环境变量
​在 backend 目录下创建 .env 文件，并填入以下必要参数：
PORT=3001
MONGODB_URI=mongodb://127.0.0.1:27017/my-vocab
JWT_SECRET=your_super_secret_jwt_key
DEEPSEEK_API_KEY=your_deepseek_api_key
CORS_ORIGIN=*
### 3. 启动服务 (推荐使用 PM2)
pm2 start server.js --name "mida-vocab"
后端启动后，静态资源将一并挂载提供服务。接着配置 Nginx 反向代理至 3001 端口即可开始使用。
